# ringTime
# 一个类似于QQ和微信的聊天工具

## 技术栈
 `uniapp` + `node.js` + `mangodb`

## 实现功能

- [ ] 好友搜索、发送好友请求;
- [ ] 个人信息查看及自己信息修改；
- [ ] 同意或拒绝好友申请；
- [ ] 建群、群管理 
- [ ] 点对点文字消息发送及接收
- [ ] 群文字消息发送及接收
- [ ] 删除好友、退出群聊
- [ ] 邮箱找回密码
